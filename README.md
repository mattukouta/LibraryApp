## 実装済み機能
### APi
WikipediaのAPI(XML)を使って、キーワードで検索した結果を表示しています。  
表示結果のリストをタップするとそのTitleのページをブラウザで開きます。  
※ 検索結果がないとアプリ終了してしまいます。(どうにかするつもりですが...)
